# Resolutions and Hilbert's Syzygy Theorem

A lecture note on free resolutions, minimal free resolutions, and projective/global dimensions, prepared as a seminar presentation for an algebraic geometry course.

## Contents

| Section | Topics |
|---|---|
| §1 Free Resolutions | Exact sequences, free resolutions, Hilbert's Syzygy Theorem, Koszul complex, Hilbert functions |
| §2 Minimal Free Resolutions | Complexes, homology, minimality, Nakayama's Lemma, uniqueness, Betti numbers via Tor |
| §3 Projective Resolutions | Projective modules, projective dimension, global dimension |

## References

- D. Eisenbud, *Commutative Algebra with a View Toward Algebraic Geometry*
- D. Eisenbud, *The Geometry of Syzygies*
- M. Atiyah & I. G. Macdonald, *Introduction to Commutative Algebra*

## Usage

Compile with any standard LaTeX distribution (e.g. TeX Live, MiKTeX):

```bash
pdflatex Hilbert-Syzygy.tex
```

Requires packages: `amsmath`, `amssymb`, `amsthm`, `tikz-cd`, `hyperref`, `geometry`, `microtype`, `xurl`, `bookmark`.
